# تطبيق تذكير الدواء

تطبيق بسيط وأنيق لتذكيرك بمواعيد تناول الأدوية الخاصة بك، يعمل كمنبه مخصص للأدوية.

## المميزات
* واجهة مستخدم بسيطة وأنيقة باللغة العربية.
* عرض قائمة بالأدوية ومواعيدها بوضوح.
* إمكانية إضافة أدوية جديدة مع تحديد الوقت المناسب للتنبيه.
* تفعيل أو إيقاف التنبيه لكل دواء بضغطة زر.
* دعم الوضع الفاتح بتصميم عصري وألوان مريحة للعين.

## كيفية التشغيل
للبدء في تشغيل التطبيق محليًا:

1. قم بتثبيت Flutter SDK.
2. قم بتشغيل الأمر التالي لجلب الحزم:
   ```bash
   flutter pub get
   ```
3. قم بتشغيل التطبيق على محاكي أو جهاز حقيقي:
   ```bash
   flutter run
   ```
